## ADDED Requirements

### Requirement: Trend Navigation
The system SHALL provide a mechanism to navigate to the PM2.5 Trend chart from the main application view.

#### Scenario: Navigating to Trend view
- **WHEN** the user presses the "Trend" button on the main screen
- **THEN** the system SHALL transition to the Trend chart screen and display the historical data

### Requirement: Trend Return Navigation
The system SHALL allow the user to return to the main screen from the Trend chart.

#### Scenario: Returning from Trend view
- **WHEN** the user presses the "Back" or "Home" button on the Trend chart screen
- **THEN** the system SHALL transition back to the main real-time data screen
